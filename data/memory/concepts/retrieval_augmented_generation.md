# Retrieval-Augmented Generation

Retrieval-Augmented Generation (RAG) is a framework that combines large language models with external knowledge retrieval mechanisms to improve accuracy and reduce hallucinations.

## Key Components

- **Retriever**: A system (often vector-based) that searches an external knowledge base for relevant documents given a query.
- **Generator**: A language model that synthesizes the retrieved information to generate a well-informed response.
- **Knowledge Base**: An up-to-date, external collection of documents, databases, or APIs.
