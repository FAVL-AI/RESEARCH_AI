# Vector Embeddings

Vector embeddings are a way of representing data as continuous multi-dimensional vectors. In NLP, they are used to capture the semantic meaning of words, sentences, or entire documents.

## Key Concepts
- **Dimensionality**: The number of continuous values used to represent a data point.
- **Cosine Similarity**: A metric used to measure how similar two embeddings are to each other.
- **Embedding Models**: Models like Word2Vec, BERT, or specialized encoder models that produce these vector representations.
